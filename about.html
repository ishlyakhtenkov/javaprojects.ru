О сайте

Добро пожаловать на сайт Java Projects!

На страницах данного сайта представлены некоторые из моих проектов, написанных на Java. Часть проектов были выполнены с целью тренировки в применении интересующих технологий, другая часть успешно введена в эксплуатацию в отделе технической документации на предприятии, где я работаю в данный момент. Пользовательский интерфейс всех приложений выполнен на английском языке, часть из них были впоследствии локализованы, однако здесь локализация недоступна.

Для каждого проекта приведена информация об использованных в нем технологиях и дано иллюстрированное описание интерфейса пользователя. Имеется возможность протестировать приложения в реальном времени и ознакомиться с их исходным кодом, выложенным на GitHub. Для скачивания доступны docker-compose файлы, позволяющие развернуть представленные проекты с помощью Docker Compose.

В настоящее время продолжаю программировать на Java, осваиваю новые технологии, закрепляю полученные знания разработкой проектов.
